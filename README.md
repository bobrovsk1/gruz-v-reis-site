# Груз в Рейс

Десктопный статический прототип веб-приложения для грузоперевозок. Интерфейс сделан как рабочий кабинет: боковое меню, расчет рейса, активные заявки, карта маршрута, вход и регистрация по телефону.

## Что внутри

- `index.html` - разметка десктопного приложения.
- `styles.css` - визуальный стиль, похожий на продуктовый логистический кабинет.
- `script.js` - расчет ставки, модальное окно входа, регистрация, SMS/Telegram demo-подтверждение.
- `languages.js` - словарь интерфейса для русского и английского языка.
- `request.html` - отдельная вкладка с шаблоном размещения заявки.
- `request.js` - сохранение черновика заявки в браузере.
- `assets/route-board.svg` - локальный визуальный ассет маршрута.
- `server.js` - простой локальный сервер без зависимостей.

## Авторизация

Сейчас реализован клиентский демонстрационный сценарий:

- вход по номеру телефона;
- регистрация по номеру, имени, роли и компании;
- выбор канала подтверждения: SMS или Telegram;
- одноразовый demo-код `2486`;
- сохранение подтвержденного профиля в `localStorage`.

Для настоящей отправки кодов нужен backend, SMS-провайдер и Telegram Bot API.

## Запуск

Можно открыть `index.html` прямо в браузере.

Для локального сервера:

```bash
node server.js
```

Адрес по умолчанию: `http://localhost:4173`.
